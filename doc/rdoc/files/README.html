Hi, this is Hobix SVN. I can’t imagine inadvertantly pulling a checkout on Hobix source code, but if you did: well, then, Hobix is a weblahhging tool. It comes with a command-line and a buncha libraries.

If you want to read all about it, see hobix.com/ for the full deal.

If you want to try Hobix out, I suggest using the online installer, which can be activated by running: 

  ruby -ropen-uri -e 'eval(open("http://go.hobix.com/").read)'

Or you can install the CVS version here by using: 

  sudo ruby setup.rb
